Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML uses tags (characters inside angled brackets) to give the information they surround special meaning

CSS

  • CSS rules appear in a separate document
  • CSS treats each HTML element as if it appears inside its own box
  • Rules are made up of selectors and declarations

JavaScript

  • JS runs in a browser by being added into an existing HTML document
  • JS makes the website more interactive

Git

  • Git status: check what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
